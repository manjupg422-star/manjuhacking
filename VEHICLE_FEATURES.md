# Vehicle Information Tool Features

## Features
- **Feature 1:** Get vehicle details by VIN
  - Usage Example: `getVehicleDetails('1HGCM82633A123456')`

- **Feature 2:** Calculate vehicle insurance premium
  - Usage Example: `calculateInsurancePremium('sedan', 2020, 'full coverage')`

- **Feature 3:** Retrieve maintenance schedule
  - Usage Example: `getMaintenanceSchedule('Toyota', 'Camry', 2020)`

## Usage Examples
1. **Fetching details for a vehicle:**
   ```
   const vehicleDetails = getVehicleDetails('1HGCM82633A123456');
   console.log(vehicleDetails);
   ```

2. **Calculating insurance:**
   ```
   const premium = calculateInsurancePremium('truck', 2018, 'liability');
   console.log(premium);
   ```

3. **Getting maintenance schedule:**
   ```
   const schedule = getMaintenanceSchedule('Ford', 'F-150', 2019);
   console.log(schedule);
   ```

## Conclusion
These features allow users to interact with vehicle data efficiently. For additional support, please refer to the documentation or reach out to support.
